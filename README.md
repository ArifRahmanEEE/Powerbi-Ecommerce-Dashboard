# E-Commerce Sales Dashboard (Power BI)

This is a Power BI dashboard built for an e-commerce sales dataset.  
It helps to quickly understand sales, profit, quantity, top states, categories, sub-categories, and payment methods.


- **Charts**
  - **Profit by Month** (monthly trend)
  - **Sum of Amount by State** (Top states)
  - **Quantity by Category** (donut chart)
  - **Quantity by Payment Mode** (donut chart)
  - **Profit by Sub-Category** (bar chart)
